Resource record DNS

Le Domain Name System (DNS) permet à un internaute de taper le nom d’un domaine dans son navigateur et d’arriver automatiquement sur le site souhaité. En réalité, la communication entre les différents éléments d’un réseau se fait au moyen d’adresses IP. Les chiffres qui composent une adresse IP étant assez complexes, il nous suffit de taper le nom du site Internet. Celui-ci est alors automatiquement traduit en une suite de chiffres.
