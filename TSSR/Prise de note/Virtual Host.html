Virtual Host

Ici “*” signifie toutes les adresse ip, on peu le remplacer pour n'avoir qu'une ip pour notre server, et le port utilisé est le “80” (1) on peu le changer également.

La ligne “ServerAdmin” (2) nous donne l'information de l'adresse mail de l'admin de notre server.

La ligne "ErrorLog" (3), nous indique comment les erreur vont être enregister, on peu le modifier selon le server par exemple (GLPI_error.log ou encore OCS_error.log)

 

Serversignature :  En "ON" te donne la version de l'os sur laquelle tourne le server (Ubuntu/debian), le port utiliser (80) la version d'apache et en “OFF” ne nous donne pas d'info.

history -c : remove l'historique des commandes qu'on a taper
