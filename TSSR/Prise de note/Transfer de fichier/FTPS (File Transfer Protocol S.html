FTPS (File Transfer Protocol Secure)

FTPS est une extension sécurisée du protocole FTP. 

Il ajoute une couche de sécurité en utilisant SSL (Secure Sockets Layer) ou TLS (Transport Layer Security) pour chiffrer les données pendant le transfert.

Cela rend FTPS plus sûr que FTP standard, car les données sont cryptées, réduisant ainsi les risques d'interception par des tiers non autorisés.
