SFTP (SSH File Transfer Protocol)

SFTP est un protocole de transfert de fichiers sécurisé qui utilise SSH (Secure Shell) pour fournir une connexion sécurisée entre le client et le serveur. 

Contrairement à FTP et FTPS, SFTP n'utilise pas deux canaux distincts (commande et données), mais plutôt un seul canal sécurisé pour toutes les communications, y compris les commandes et le transfert de données. 

SFTP offre une sécurité et une fiabilité élevées, en utilisant le cryptage pour protéger les données et les informations d'authentification.
