TFTP (Trivial File Transfer Protocol)

TFTP est un protocole simplifié utilisé pour transférer des fichiers entre des machines sur un réseau. 

Port 69 udp

Pas de listage de fichier pas de notion de lecture / écriture

Il est "trivial" dans le sens où il ne nécessite pas d'authentification ou de sécurité intégrée.

Il est souvent utilisé pour le démarrage à distance des périphériques réseau, tels que les routeurs et les commutateurs, car il est léger et rapide.
