POE (Power Over Ethernet) : est une technologie de réseaux locaux (LAN) Ethernet filaires qui fait passer le courant électrique nécessaire au fonctionnement de chaque appareil par les câbles de données, au lieu des cordons d'alimentation.

BYOD (Bring Your Own Device/ Apportez Votre Equipement personnel de Communication) : qui désigne l'usage d'équipements informatiques personnels dans un contexte professionnel.
Exemple de topologie : BUS / Etoile / Etoile étendue / Ring

LAN : Réseau Local (Ethernet)
WLAN : Réseau Local sans file (802.11 x) 

Ce sont des réseaux de type LAN qui permettent de s’affranchir de l’usage des câbles pour connecter les périphériques. Ils sont, par abus de langage, souvent qualifiés de réseaux Wi-Fi, car ils se conforment aux normes édictées par la Wi- Fi Alliance {WIreless- FIdelity} dont l’adresse du site esthttp://www.wi-fi.org/.
Les protocoles réseaux locaux sans fils sont 802.11a, 802.11b,802.11g , 802.11net 802.11ac.
Pourquoi 802 ? Parce que quand elle à été inventée, c'était en l'an 19(80) au moi de Février (2)

MAN : {Metropolitan Area Network} sont des réseaux dont l’étendue couvre des zones dont la taille varie de celle d’un campus à celle d’une communauté urbaine.

WAN {Wide Area Network, WAN} : Quelque chose qui permet de connecté plusieurs réseau locaux entre eux. Ces ordinateurs étaient implantés sur des sites géographiques disséminés sur l’ensemble du territoire des USA. ARPAnet constitua ainsi le premier réseau étendu. 

PAN : Personnel, Bluetooth. Il permet de relier des équipements "très" proches entres eux. Bluetooth, portée 100 mètres, et ZigBee portée 10 mètres sont des PANs sans fil, tandis que USB {Universal Serial Bus} est un PANfilaire.

TCP/IP : {Transmission Control Protocol / Internet Protocol} qui contrairement aux réseaux propriétaires, permet de connecter des équipements de structures diverses par l’intermédiaire d’un réseau public : Internet.

 

PDU/UDP :(Protocole Data Unit / Unité de Data Protocolaire) : La description des données utilisé par le protocole.
Profil de connexion réseau : Publique / Privée
 
Plus on descend dans OSI plus on segmente la PDU
Le principal d'encapsulation donne un en tête et un en queue a chaque PDU.
En tête : Info sur la source / Info sur la destination
En queue :

  1. Physique : Répéteur / Concentrateur (HUB) (Représentation des bits sur les supports) PDU : bits
  2. Liaison de données : MAC (Politique d'accès au média)(Pour être capable de faire discuter des équipement adjacent) Protocole : Token Ring / Ethernet / PPP / PPPOE. (On ne peux faire parler entre eux que des équipement qui utilise le même protocole) PDU : Trame
  3. Réseau : PDU : Packet
  4. Transport : TCP/UDP PDU : Segment
  5. Session : PDU : Données
  6. Présentation : PDU : Données
  7. Application : PDU : Données
