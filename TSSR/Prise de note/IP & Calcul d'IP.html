IP & Calcul d'IP

/32 = 1 /31= 2 /30 = 4 /29 = 8 /28 = 16 /27 = 32 /26 = 64 /25 = 128 /24 = 256 /23 = 512 /22 = 1024 /21 = 2048

 

Exemple d'adresse IP privées (RFC 1918) :
192.168.255.1 / 172.25.255.1 / 10.30.81.52

Une adresse APIPA commence TOUJOURS par 169 :
Exemple : 169.254.19.3

Une adresse IP de boucle locale (loopback) commence TOUJOURS par 127 :
Exemple : 127.0.0.1
