NBNS

Le NBNS aussi appeler le “Netbios Name server” est principalement utiliser par Microsoft. 

C'est un system de nommage, il fait la conversion d'un nom d'ordinateur en adresse IP. 

Sur le systèmes Windows “NBNS” est connu sous le nom “WINS” (Windows Internet Names Service).

NBNS tient un registre de tous les participants sur un réseau. Ce qui lui permet d'envoyer des requêtes rapide afin d'établir si un nom demandé est déjà réservé . 

Bien que le NBNS soit utilisé pour résoudre les nom Netbios, il ne doit pas être confondu avec le DNS, qui lui est utilisé pour résoudre les noms de domaine en adresse IP.
