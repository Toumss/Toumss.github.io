Active Directory (AD)

Active directory c'est quoi ?

Un annuaire qui repertories et organise les information du monde microsoft :

  • les utilisateur 
  • Les machines 
  • Les application

Sont objectif est de centraliser deux fonctionnalités essentielles :

  • L'identification
  • Authentification

Au sein d'un system d'exploitation.

A quoi ce sert ?

L'active Directory a 4 grand point d'intérêts :

  • Ca permet a l'administration de se centralisé et de se simplifier, car elle regroupe tout au seins d'une même base.
  • Unifier les authentifications. Exemple un utilisateur authentifier sur une machine du domaine, qui elle même est authentifier sur le domaine auprès d'un contrôleur de domaine, donne access à l'utilisateur à différente resources (Un server de fichier, une application, une imprimante etc…)
  • Identifier les objets sur le réseau car chaque objet enregistré dans l'annuaire est unique ce qui permet de l'identifier facilement et de savoir quel ordinateur a accéder a quelle ressource et à quel moment grace au log.
  • Référencer les utilisateurs et les ordinateurs.
