Installation agent Linux

Il faut commencer par aller chercher le packet “Veeam” .deb (veeam-release-deb_1.0.8_amd64.deb), une fois télécharger il faut le copier sur notre distribution linux puis utiliser la commande suivante :

dpkg - i veeam-release-deb_1.0.8_amd64.deb
apt update
apt install veeam
apt install blksnap
apt install veeamsnap

 

Une fois fais on peut taper la commande :

veeam

Le logiciel ce lance.
