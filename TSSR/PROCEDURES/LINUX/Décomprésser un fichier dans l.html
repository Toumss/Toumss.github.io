Décomprésser un fichier dans linux

Trouver le fichier dans le terminal linux que l'on veux décompresser le copié “cp” si besoin dans “usr/src” puis éxécuter cette commande  pour le décompréssé.

tar -xf filename.tar.gz

Pour copier le fichier qui est dans “Utilisateur” vers “root ce connécté en root puis tapé la commande 

cp /home/nom_utilisateur/nom_du_fichier
