Installer Docker

Pour etre sur qu'il n'y a pas une ancienne version :

apt-get remove docker docker-engine containerd runc docker-ce

Installer docker :

apt update
apt install docker.io docker-compose -y

On ajoute un utilisateur docker au groupe :

usermod -aG docker debian

Puis on redemarre l'OS pour que ca soit pris en compte 

reboot

Pour voir mes images de conteneur :

docker ps -a
