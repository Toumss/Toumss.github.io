Installer/Supprimer un conteneur

Ici pour l'exemple on utilisera un conteneur ubuntu :

docker search ubuntu
docker pull ubuntu
docker image
docker start "mon-ubuntu"
docker exec -it "mon-ubuntu" /bin/bash

Ici on est passé sur le Ubuntu

Pour sortir de notre conteneur ubuntu :

exit

pour supprimé notre conteneur Ubuntu :

On commence par le stopper :

docker stop "mon-ubuntu" (NAMES)
docker rm "mon-ubuntu" (NAMES)

Pour supprimé notre image de conteneur (iso) :

docker rmi "ubuntu" (nom de l'image)
