Ajouter et monter un HDD

  1. Passé en mode root / mdp
su -

2)  Pour voir les disques présent sur la Debian

fdisk -l

3)  Pour crée une partition sur le disque

cfdisk /dev/sdX

=> gpt => Nouvelle => Ecrire => Oui => Quitter

4) 

mkfs.ext4 /dev/sdXN


=> Yes/no = Y

5) Crée le dossier ou monter le hdd
Se rendre dans /mnt puis crée le dossier voulu avec (mkdir ”nom du dossier")

6) Blkid (Récupere l'UUID de ta partition de ton disque dur)

nano /etc/fstab


UUID=ffedaa2c-94d3-4565-a03b-94948ae852f1        /mnt/data ext4 rw,user,auto (ou default)    0       2
ctrl-x (quitter) / sauvergarder ? O/Y / fstab… ? “ENTRER”

8) mount -a (pour monter le disque) (daemon si nécéssaire)
