REMBG

REMOVE BACKGROUND

 

On install pip

apt install python3-pip

 Puis on install venv

apt install python3-venv

On se crée un dossier venv nommer “venv”

python3 -m venv venv

On entre dans notre environnement venv

source venv/bin/activate

Une fois dedans on install notre programme REMBG

pip install rembg
pip install rembg[gpu,cli]
pip install onnxruntime

On copie notre photo de notre répertoire jusqu'a notre environnement :

 cp /home/debian/Bureau/lili.jpg .

Puis on peu utiliser notre commande pour enlever le fond comme ceci 

(rembg i “notre_photo.jpg” "notre_photo_modifier.jpg)

rembg i lili.jpg lili1.jpg

Une fois fait il nous reste plus qu'a move notre photo de notre environnement a notre repertoire 

mv lili1.jpg /home/debian/Bureau/
