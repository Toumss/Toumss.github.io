Mysql / Mariadb

Dans son server linux taper la commande :

apt install mariadb -server

Pour crée une base de donnée utiliser la commande :

CREATE DATABASE nomdeladatabase ;

Pour crée un nouvel utilisateur utiliser la commande :

CREATE USER 'nouveau_utilisateur'@'localhost' IDENTIFIED BY 'mot_de_passe';

Pour donner tous les privilèges a u utilisateur utiliser la commande :

GRANT ALL PRIVILEGES ON * . * TO 'nouveau_utilisateur'@'localhost';

Pour voir les databases présente sur mariadb utiliser la commande :

show databases ;

Pour entrer dans une databases utiliser la commande :

use nomdeladatabase

Pour voir les utilisateur qui ont des priviliege dans cette database utiliser la commande :

show grants ;

 

Pour integrer un utilisateur dans une base de domaine il faut lui donner des privilèges dans cette database a l'aide de la commande :

GRANT ALL PRIVILEGES ON Nintendo.* TO 'mario'@'tssr.local';
