TUIFIMANAGER

Pré-requis

Pour installer et utiliser “Tuifimanager” on commence dabord par se mettre en root 

su -

Puis :

apt update && sudo apt upgrade -y

Ensuite on install python3

apt-get install python3-pip

On utilise cette commande pour éviter les problèmes éventuelle :

rm /usr/lib/python3.*/EXTERNALLY-MANAGED

Puis on install télécharge et install “TuifiManager” :

pip3 install tuifimanager --upgrade

Il ne nous reste plus qu'a taper la commande suivante pour ouvrir le logiciel :

tuifi

 

Felicitation vous avez un explorateur de fichier dans votre terminal !
