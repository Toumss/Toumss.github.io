Créer un alias ssh

Créer un alias srv1 sur le PC Windows pour se connecter directement avec la clé de chiffrement SSH : 

  • Mettre la clé privée dans C:\Users\Candidat\.ssh
  • Créer un fichier config (sans extension) dans le dossier C:\Users\Candidat\.ssh

Copier les informations pour l’alias dans le fichier config : 

Host srv1               # You will use it to login

HostName 192.168.240.147         # host domain orip

Port 22              # ssh port

IdentitiesOnly yes            # Use ssh key or not

IdentityFile C:\Users\Candidat\.ssh\ssh1   # Path of pub key

User debian                 # (選填)登入 SSH 的 username，

ForwardX11 yes
