Intégration agent

Pour importer son ordi en tant qu'agent dans OCS on commence a télécharger l'agent :

https://github.com/OCSInventory-NG/WindowsAgent/releases

 

Une fois télécharger on lance l'installation, une fois arrivé sur cette fenêtre on indique l'adresse de son server avant "/ocsinventory" (1) , puis on clique sur “Suivant” (2) :

 

On clique sur suivant jusqu'a arrivé sur cette page puis on active “Enable verbose log” (1), ainsi que "Immediately launch inventory (= /NOW) (2), puis on clique sur suivant et on fini l'installation  :

 

http://192.168.240.138/ocsreports/
