Iptables

Pour l'installer :

apt install iptables

Ou pour que ca soit persistant :

apt install iptables-persistent

Pour coupé l'accés a un port :

iptables -D INPUT -p tcp --dport 80 -j DROP

le -D/A : Delete une règle / Ajoute une règle 

Input : Dans ton réseau

Output : Hors de ton réseau

-p : le protocole ici tcp

- dport : port de déstination

- sport : port source

-80 ici on bloque le port 80

-j indique l'action a faire ici DROP/ACCEPT
