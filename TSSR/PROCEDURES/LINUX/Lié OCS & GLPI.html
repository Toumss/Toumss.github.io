ACTIVER LES PLUGINS GPLI

 

On se rend sur l'interface graphique GLPI, puis dans l'onglet “Configuration” (1) , “Plugins” (2), et on active les plugins (3) :

TELECHARGEMENT / INSTALLATION 

On tape la commande suivante :

apt install git

On se rend dans le repertoire suivant :

cd /var/www/html/glpi/plugins

Et on télécharge le plugins :

wget https://github.com/pluginsGLPI/ocsinventoryng/releases/download/2.0.4/glpi-ocsinventoryng-2.0.4.tar.bz2

Et on décompresse l'archive :

tar -xf glpi tab
