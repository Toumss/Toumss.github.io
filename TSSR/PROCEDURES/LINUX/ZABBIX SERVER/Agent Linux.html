Agent Linux

On commence par aller chercher le paquet :

wget https://repo.zabbix.com/zabbix/6.4/debian/pool/main/z/zabbix-release/zabbix-release_6.4-1+debian12_all.deb

On le décompresse :

dpkg -i zabbix-release_6.4-1+debian12_all.deb

On met  jour :

apt update

On install l'agent :

apt install zabbix-agent -y

On lance l'agent et on fait en sorte qu'il se lance au boot 

systemctl restart zabbix-agent

systemctl enable zabbix-agent

Ce rendre dans le fichier .conf de zabbix pour renseigner notre ip server zabbix :

nano /etc/zabbix/zabbix_agentd.conf

 

 

On descent encore pour chercher “ServerActive” on y renseigne également notre ip du Server Zabbix : (Ctrl-w pour recherche rapide)
