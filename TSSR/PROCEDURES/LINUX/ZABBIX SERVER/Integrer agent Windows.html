Integrer agent Windows

Integrer l'agent dans les hôte voulu :

Télécharge l'agent :

https://www.zabbix.com/fr/download_agents?version=6.4&release=6.4.10&os=Windows&os_version=Any&hardware=amd64&encryption=OpenSSL&packaging=MSI&show_legacy=0

 On lance le logiciel sur la machine que l'on veux surveiller, arrivé à cette étape on indique l'ip de notre server zabbix (1) également sur la ligne “Server or proxy…”, puis on clique sur “Next” (2), puis on l'install :

 

Une fois installer on se rend dans les pare-feu windows de notre machine (2) , puis no clique sur “Autoriser une application ou une fonctionnalité via le Pare-feu Windows Defender” (1) :

 

 

On vérifie que Zabbix agent soit bien coché avec toutes les autorisation (1) , puis on clique sur “Autoriser une autre application…” (2) :

 

Cette fenêtre s'ouvre on clique sur “Parcourir” (1) :

 

Puis on va chercher le chemin jusqu'à (C:\Program Files\Zabbix Agent) “zabbix_agentd” (1) , on le séléctionne (2), puis on clique sur “ouvrir” (3) :

 

Une fois ajouter on lui donne tous les droits (1), puis on clique sur “ok” (2) : 

 

On retourne sur notre Zabbix on clique sur “surveillance” (1), puis sur “crée un hôte” (2) :

 

Arrivé sur cette page on indique le nom de l'hôte que l'on veut surveiller (1) , dans les modèles on clique sur “sélectionner” (2) puis on choisis “Windows by zabbix agents”, dans les groupes d'hôtes on clique sur “sélectionner” (3) puis on choisis “Discovered hosts” pour finir dans interfaces, on clique sur “ajouter” (4) puis on indique l'adresse ip de l'hôte en question, on peux enfin cliquer sur “Ajouter” en bas à droite.
