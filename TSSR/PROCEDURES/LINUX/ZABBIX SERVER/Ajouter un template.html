Ajouter un template

Pour ajouter un template on se rend dans “Collecte de donnée” (1) puis “modèle” (2), on clique sur "importer" (3) puis on clique sur “choisir un fichier” (4) pour choisir son fichier xml.

 

Par exemple, pour un template wsus, ce rendre sur https://github.com/rozniak/Zabbix-WSUSReporting/blob/master/Template%20WSUS%20Server.xml 

Télécharger le fichier XML. 

Pour le rendre effectif sur notre server on se rend dans l'onglet “Surveillance” (1) , on clique sur “Hôtes” (2), puis on choisi le server auquel on veut rajouter la template (3), et on se rend dans "hôte" (4) :

 

 

Une nouvelle fenêtre s'ouvre dans l'encadré on clique sur “Sélectionner” (1) , pour y choisir notre template (2)
