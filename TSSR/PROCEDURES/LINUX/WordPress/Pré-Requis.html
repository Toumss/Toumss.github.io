PRÉ-REQUIS

 

Toutes les commandes doivent être entrer en “root”

On commence “mettre à jour” les paquet sur la machine Debian 12 :

apt-get update && sudo apt-get upgrade

Ensuite on s'attaque au “pré-requis” avant l'installation de GLPI, c'est a dire “Apache2”, php et mariadb-server :

apt-get install apache2 php mariadb-server

On install PHP 8.2 :

apt-get install php8.2 php8.2-cli php8.2-common php8.2-imap php8.2-redis php8.2-snmp php8.2-xml php8.2-mysqli php8.2-zip php8.2-mbstring php8.2-curl libapache2-mod-php -y

On crée un Utilisateur avec un mot de passe pour Wordpress :

mysql -U -p
CREATE USER 'wordpress'@'localhost' IDENTIFIED BY 'YourStrongPasswordHere';

On crée la DataBase :

 CREATE DATABASE wordpress;

On donne les privileges a notre utilisateur :

 GRANT ALL PRIVILEGES ON wordpress.* TO 'wordpress'@'localhost';

Puis on refresh les privileges :

FLUSH PRIVILEGES;

Puis on exit :

exit
