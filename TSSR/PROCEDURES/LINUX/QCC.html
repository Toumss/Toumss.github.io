QCC

Dans un ETC on a tous les fichier de configuration en fichier texte

Dans BOOT tous ce qui est démarage

Le ROOT c'est tous ce qui concerne le super utilisateur

HOME fichier personelle utilisateur

Pour modifier la liste des dépots

/etc/apt/source.list

 Pour gerer les privilieges des utilisateur

/etc/sudoers

Pour crée un fichier vide :

touch

Pour télécharger un élément : 

wget

Lancé un service au boot : 

systemctl enable "NomDuService"

supprimer un packet : 

apt purge "NomDupacket"
