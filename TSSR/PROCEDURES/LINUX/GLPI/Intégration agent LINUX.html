Intégration agent LINUX

On met a jour :

apt update && apt upgrade -y

On télécharge l'agent :

wget https://github.com/glpi-project/glpi-agent/releases/download/1.4/glpi-agent-1.4-linux-installer.pl

On installe perl :

apt install perl

On install l'agent avec perl :

perl glpi-agent-1.4-linux-installer.pl

https://10.10.10.11/

On nous demande de taper l'adresse ne notre server GLPI

glpi-agent

 

Si probleme,  faire la commande :

cd /etc/glpi-agent/conf.d
nano 00-install.cfg

 

EASY GAME !
