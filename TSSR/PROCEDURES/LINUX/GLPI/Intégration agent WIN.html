Intégration agent WIN

Pour importer son ordi en tant qu'agent dans GLPI on commence a télécharger l'agent :

https://github.com/glpi-project/glpi-agent/releases/tag/1.6.1

Une fois télécharger on lance l'installation :

On fait une installation typique, “next” “next” “next” arrivé à cette fenêtre dans “Remote Targets” (1) on indique notre adresse ip accompagner de “http://” et finir par “/glpi” (ex : http://192.168.240.138/glpi),puis on clic sur “next” (2) :

 

On oublie pas dans notre interface graphique GLPI d'activé l'inventaire en passant par l'onglet “Administration” (1), puis “Inventaire” (2), et on active l'inventaire (3), (on oublie pas de sauvegarder en bas de la page) :

 

Et pour finir pour faire entrer notre agent dans GLPI on se rend sur une page web et on entre “http://localhost:62354/”, on clic sur “Force an inventory” (1) et si ca se passe bien on a “The current status is running task inventory” (2) :
