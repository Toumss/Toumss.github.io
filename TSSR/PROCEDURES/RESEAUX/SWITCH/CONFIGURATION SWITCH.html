CONFIGURATION SWITCH

Une fois la connexion faire on doit arrivé sur cette page :

(Si on nous propose quelque chose (Yes/No) dire No puis entrer)

Ensuite on tape “enable” comme ca on peux vérifier si le switch dispose d'un mot de passe. 
Dans ce cas la trouver le bouton "mode" du switch puis laisser appuyer dessus jusqu'a reinitialisation :

On commencera par changer le nom de notre appareil :

conf t
hostname "Nom"

 

Pour ajouter un mot de passe chiffrer :

enable secret

 

On vient ensuite configurer l'ip de la Vlan1 de notre switch :

conf t
interface vlan1
ip address "0.0.0.0"(Adress ip) "0.0.0.0" (Masque)

 

On lui donne un domaine :

ip domain-name "NomDeDomaine"

 

Pour acceder au SSH on commence par donner la version utiliser :

ip ssh version 2

 

Ensuite on crée une clef rsa :

crypto key generate rsa

Puis on tape 1024

 

On crée un utilisateur avec un mdp crypté

username toto secret toto

 

Ensuite on crée les ligne ssh :

conf t
line vty 0 4

 

On authorise les login local :

line 0 4
login local
transport (en fonction du switch) input ssh / preferred ssh

 

Pour garder notre configuration même après un redémarrage on utilise la commande :

write memory

ou 

copy running-config startup-config
