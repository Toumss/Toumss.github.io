Configurer Switch

A partir d'un switch sur packet tracer (1), ce rendre dans le CLI (2) puis entrer les commandes suivantes :

 

 

Avant tout :

Switch>enable
Switch#configure terminal
Enter configuration commands, one per line, End with CNTL/Z

Pour changer le hostname du Switch taper la commande :

hostname "SW-1"

Pour crée les VLAN taper les commandes suivante :

SW-1(config)#vlan 10
SW-1(config)#name VLAN10
SW-1(config)#vlan 20
SW-1(config)#name VLAN20
SW-1(config)#end

Pour rediriger les ports en fonction des VLAN taper les commandes suivantes :

SW-1#configure terminal
SW-1(config)#interface range FastEthernet 0/1-7
SW-1(config-if-range)#switchport mode access
SW-1(config-if-range)#switchport access vlan 10
SW-1(config-if-range)#switchport nonegociate
SW-1(config-if-range)#exit

 

Pour faire communiquer deux switch qui possède 2 réseaux en commun il faut connecter le switch (1) en mode trunk, comme l'autre est automatiquement en mode dynamic auto cela fonctionnera, mais le mieux et de mettre le switch (2) en mode trunk aussi :

 

Pour changer le mode entre (acces, trunk, mode dynamic désirable, dynamic auto…) :

SW-1#configure terminal
SW-1(config)#interface "g0/1" 
SW-1(config)#switchport mode "trunk/acces/dynamic desirable..."
SW-1(config)#exit

 

Pour que l'interface d'un switch en mode trunk n'autorise que certaine VLAN taper les commandes suivante :

SW-1#configure terminal
SW-1(config)#interface "g0/1"
SW-1(config)#switchport trunk allowed vlan 10,20 (...)
SW-1(config)#exit
