Configuration routeur on a stick

Pour configurer le routeur (1) afin qu'il puisse intégrer les 3 réseau sur la même interface (2) on met en place 3 sous interface virtuelle sur l'interface sur laquelle il est connecter, sur chaque interface virtuelle on y ajouter une adresse ip qui servira de passerelle par défaut au poste sur chaque réseaux différent :

 

Routeur>enable
Routeur#configure terminal
Routeur(config)#interface g0/0.10
Routeur(config-subif)#encapsulation dot1Q 10(20,30)
Routeur(config-subif)#ip address 172.31.10.254 255.255.255.0
Routeur(config-subif)#exit
Routeur(config)#interface g0/0.20
Routeur(config-subif)#encapsulation dot1Q 20(10,30)
Routeur(config-subif)#ip address 172.31.20.254 255.255.255.0
Routeur(config-subif)#exit
Routeur(config)#interface g0/0.30
Routeur(config-subif)#encapsulation dot1Q 30(10,20)
Routeur(config-subif)#ip address 172.31.30.254 255.255.255.0
Routeur(config-subif)#exit

Ne pas oublier d'allumer l'interface !

Routeur(config)#interface g0/0
Routeur(config-if)#no shut
