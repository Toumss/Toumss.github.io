Configuration Switch niv.3 (MLS)

Quand on met en place un switch de niveau 3 (MLS) (1) qui est donc capable de faire du routage, on commence déja par connécter les switch déja éxistant sur les interfaces voulu (2) (3) (4) qu'on va passé en mode trunk pour quelle puisse communiquer de trunk à trunk :

Switch>enable
Switch#configure terminal
Enter configuration commands, one per line.  End with CNTL/Z.
Switch(config)#ip routing
Switch(config)#interface G 1/0/1
Switch(config)#ip address 192.168.10.254 255.255.255.0
Switch(config-if)#switchport nonegotiate
Switch(config-if)#switchport mode trunk
Switch(config-if)#exit
Switch(config)#interface G 1/0/2
Switch(config)#ip address 192.168.20.254 255.255.255.0
Switch(config-if)#switchport nonegotiate
Switch(config-if)#switchport mode trunk
Switch(config-if)#exit
Switch(config)#ip address 192.168.10.254interface G 1/0/3
Switch(config)#ip address 192.168.30.254 255.255.255.0
Switch(config-if)#switchport nonegotiate
Switch(config-if)#switchport mode trunk
Switch(config-if)#exit

Ensuite on crée nos vlan qui sont déja éxistante sur les autre switch :

Switch#conf t
Enter configuration commands, one per line.  End with CNTL/Z.
Switch(config)#vlan 10
Switch(config-vlan)#name VLAN10
Switch(config)#vlan 20
Switch(config-vlan)#name VLAN20
Switch(config)#vlan 30
Switch(config-vlan)#name VLAN30

Une fois fais bien vérifier que les interfaces des switch qui son connecter ai MLS soit bien en mode trunk (1) (2) (3) :

 Sur chaque routeur taper les commandes suivante :

SW-1#configure terminal
SW-1(config)#interface "g0/1" 
SW-1(config)#switchport mode trunk
SW-1(config)#exit
