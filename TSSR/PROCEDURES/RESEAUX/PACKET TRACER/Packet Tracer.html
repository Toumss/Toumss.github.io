Packet Tracer

POUR LE PING

Requête ARP : Envois à tous le monde une requête (Trame) (Broadcast), pour connaitre l'adresse MAC de l'appareil de destination, à partir de son ladresse IP.

Commutateur (Switch): il batti sa table de commutation en fonction des flux qui le traverse.

ping - t : Ping a durée indérterminé
 

(Commande : arp -a) = connaitre sa table/cache arp
(Commande : arp -d) = Reset sa table arp

Requête ICMP : Protocole qui permet de diagnostiquer et savoir si il y a une erreur sur le réseau.

 

DANS LE SWITCH “Onglet CLI” :

Commands : 

? = Connaitre toutes les commandes


show mac-address-table : Voir la table d'adress mac et sur quel port il sont couplé.


Il est possible d'administrer une adresse ip à un switch, pour faire de l'administration à distance, ou pour vérifier à l'aide d'un ping la disponibilité du switch.

DANS LE ROUTEUR :

Configurer une interface :

onglet CLI, ctrl + Z (#routeur), “?” pour avoir les commands

configure terminal (routeur config)

interface g0/0 (routeur‼ config-if)

ip adress (ex: 172.16.255.254 255.255.0.0)

no shut down (allumer l'interface)

show ip interface b (Pour voir les configuration de nos interface)
