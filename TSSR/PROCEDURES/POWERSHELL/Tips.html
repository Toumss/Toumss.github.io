Tips

Pour ajouter un onglet de script :

Ctrl+n

Pour trouver une commande :

Get-Command -Name *ip*address*

Pour connaitre le nom de notre interface réseau :

Get-NetConnectionProfile

Retiré une adresse IP :

Remove-NetIPAddress -IPAddress "000.000.000.000"

Ajouter une adresse IP :

New-NetIPAddress -IPAddress 172.31.0.2 `

                 -PrefixLength 16 `

                 -InterfaceIndex 3 `

                 -DefaultGateway 172.31.0.254
Set-DnsClientServerAddress -InterfaceIndex 3 -ServerAddresses "8.8.8.8" , "1.1.1.1"
