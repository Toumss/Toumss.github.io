Instaurer des quotas

1) Commencer par ajouter un service dans le serveur “Gestionnaire de ressources du serveur de fichier” (1), puis suivant (2) :

 

2) Dans le “Gestionnaire de ressources du serveur de fichier” clic sur “Gestion de quota” (1), puis modèle de quotas (2) :

 

3) Une fois dans “modèle de quotas”, clic droit puis “créer un modèle de quota…” (1) :

 

4) Donner le “Nom du modèle” (1), puis la “Limite d'espace” (2), et finalement “OK” (3) :

 

5) Pour crée le quota, Gestionnaire de ressources du serveur de fichier, puis clic sur “Quotas” (1), puis “Créer un quota…” (2) :

 

6) Choisir le chemin d'accès pour le quota “Parcourir” (1), selectionner le fichier (2), puis “OK” (3) :

 

7) Une fois le chemin d'accès choisi, on peu “Délivrer les propriétés de ce modèle de quota” (1) et choisir dans le menu déroulant le quota souhaiter (2), puis clic sur “créer” (3) pour appliquer le quota :
