Ajouter un utilisateur

Copier le fichier de base pour pas le perdre 

cp /etc/asterisk/sip.conf /etc/asterisk/sip.conf.original

Vider le fichier sip.conf 

> /etc/asterisk/sip.conf

On édite le fichier  

nano /etc/asterisk/sip.conf


(Pour pouvoir tester comme il ce doit créer une utilisateur 101)

 

Faire une copie du fichier extensions 

cp /etc/asterisk/extensions.conf /etc/asterisk/extensions.conf.original

On vide le fichier  

> /etc/asterisk/extensions.conf

On édite le fichier 

nano /etc/asterisk/extensions.conf

(tout ce qui se trouve après « ; » est un commentaire. Il n’est donc pas obligatoire de le mettre.
Le motif « _1XX » permet de prendre en compte tous les numéros allant de 100 à 199)

Aller dans la console d’asterisk pour voir les utilisateurs créer  

asterisk -rvv ; (ouvrir la console)

Pour recharger pour prendre en compte les changements

reload ;

Permet de voir les utilisateurs créés :

sip show users ;
