Config messagerie

Passer un appel simple avec messagerie

  1. Editer le fichier extensions 
Nano /etc/asterisk/extensions.conf

  1. Configuration du VoiceMail
  2. On copie le fichier original  
cp /etc/asterisk/voicemail.conf /etc/asterisk/voicemail.conf.original

On vide le fichier  

> /etc/asterisk/voicemail.conf

Et on édit le fichier  

nano /etc/asterisk/voicemail.conf

 

Exemple :

[general]

format=ulaw|alaw
attach=yes
servermail= Test@default
emailsubject= Nouveau message de ${VM_CIDNAME}
emailbody= \n\tBonjour ${VM_NAME}, \n\n\tTu as un message de la part de ${VM_CIDNAME} d une duree de ${VM_DUR} datant d>maxmsg=20
maxsecs=30
minsecs=2
maxlogin=3
review=yes
saycid=yes

[default]

101= 123, Thomas, 101@default ; 101
102= 123, Vincent, 102@default ; 102

Reload les paramètres dans la console Asterisk  

asterisk -rvv ; ouvrir la console

Pour recharger pour prendre en compte les changements 

reload ;
