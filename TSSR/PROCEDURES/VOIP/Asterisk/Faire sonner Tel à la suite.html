Faire sonner Tel à la suite

Modifié le fichier extensions 

nano /etc/asterisk/extensions.conf

[default]

[local]

exten => _1XX, 1, Answer ()
exten => _1XX, n, Dial(SIP/${EXTEN},10)
exten => _1XX, n, VoiceMail(${EXTEN},10) 
exten => _1XX, n, Hangup()

exten => 888, 1, VoicemailMain (${CALLERID(num)}); n° de boite vocal, on peut mettre le numéro que l'on veut
