Pour ajouter un contexte

Ce rendre dans le fichier "extensions.conf"

nano /etc/asterisk/extentions.conf

Donner un nom au contexte ici : [MECA]

Premier groupe de commande (1) :

1) Ouvre la ligne
2) Compose le numéro, fait sonné 10sec
3) EN cas de non réponse renvois sur boite mail
4) Ferme la ligne 

Deuxième commande (2) : Numéro a composé pour boite mail

Troisième commande (3) : Si un numéro entre 400 et 499 et composé renvois au contexte “it” 

 

Exemple :

 

[COMPTA]

exten => _1XX, 1, Answer()
exten => _1XX, n, Dial(SIP/${EXTEN},10)
exten => _1XX, n, Playback(standard)
exten => _1XX, n, VoiceMail(${EXTEN},10)
exten => _1XX, n, Hangup()

exten => 789, 1, VoicemailMain(${CALLERID(num)})

exten => 1000, 1, Dial(SIP/101&SIP/102,10)
exten => 1000, n, Hangup()
exten => 11,1,Goto(IVR,11,1)
