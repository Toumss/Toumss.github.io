Conférence

Crée un contexte “Conférence” :

[CONFERENCE]

exten => 100, 1,Answer()
exten => 100, n,Dial(SIP/100) 
exten => 100, n,Hangup() 


exten => 200, 1,Answer()  
exten => 200, n,Dial(SIP/200) 
exten => 200, n,Hangup() 


exten => 666,1,Progress() ; le numero 666 permet aux invités de se connecter (password 1234) 
exten => 666,2,Wait(1) 
exten => 666,3,ConfBridge(1,default_bridge,default_user) 


exten => 777,1,Progress() ; le numero 777 permet à l'administrateur de se connecter (password 1234) 
exten => 777,2,Wait(1) 
exten => 777,3,ConfBridge(1,default_bridge,admin_user)

Se rendre dans “confbridge” pour y coller le texte qui suis :

nano /etc/asterisk/confbridge.conf

 Puis copier-coller le texte suivant : 

[general]

[admin_user]

type=user ; permet d'appeler / recevoir
pin=1234 ; pin pour intégrer la conférence 
marked=yes ; je sais pas trop 
admin=yes ; c'est lui qui administre
music_on_hold_when_empty=yes ; musique d'attente pendant que la conférence est vide
announce_user_count=yes ; annonce le nombre d'utilisateurs connectés sur la conférence
 

[default_user]

type=user ; permet d'appeler / recevoir
pin=1234 ; pin pour intégrer la conférence
wait_marked=yes ; Met l'utilisateur en attente
end_marked=yes ; previent qu'il quitte la conférence
music_on_hold_when_empty=yes ; met une musique d'attente si l'administrateur n'est pas en ligne
announce_user_count=yes ; annonce le nombre d'utilisateurs connectés sur la conférence

[default_bridge]

type=bridge ; permet de définir une conférence
max_members=10 ; nombre d'utilisateurs maximum

Puis sauvegarder (ctrl + o) et quitter (ctrl + x)
