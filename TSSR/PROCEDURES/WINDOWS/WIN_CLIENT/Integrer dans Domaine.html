Integrer dans Domaine

Pour intégrer sa machine dans un domaine, suivre la procédure  Changer le nom de la machine

 

  1. Au moment de changer le nom de l'ordinateur, clic sur “Domaine” (1), puis “OK” (2), une fenêtre s'ouvre demandant un compte et un mot de passe a renseigner. (ex : “Compte : TSSR\Administrateur” "Mdp : Greta37!)  

 

2) Mettre l'ip du server en DNS sur machine qui veux rejoindre le domaine
