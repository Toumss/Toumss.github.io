1) Avant de rentré le Windows Client dans le domaine se rendre dans la barre de recherche et taper “Gérer les fonctionnalités facultatives” (1), puis clic sur “Gérer les fonctionnalités” (2) :

 

2) Clic sur "ajouter une fonctionnalité (1) :

 

3) Puis installer les fonctionnalités voulu “Gestionnaire de serveur” (1), “Outils Active Directory Domain service” (2), “Outils du serveur DHCP” (3), “Outils du serveur DNS” (4) :

 

4) Pour accéder au centre de Gestion du serveur, cliqué sur “Démarrer” (1), scroll jusqu'a “Outils d'administration Windows” (2), puis “Centre d'administration Active directory” (3) :
