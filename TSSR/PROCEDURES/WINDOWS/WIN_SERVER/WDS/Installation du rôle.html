Installation du rôle

WDS sert à déployer des images windows a partir de fichier “.wim”

Au préalable on ajoute et on monte un second disque dur.
(Pour y installer nos images)

A partir de la sélection des rôle server (1) , on commence par intégrer le role DNS (on configure le dns inversé, DHCP (On le configure avec une étendu) , AD DS ainsi que “Service de déploiement” (2) sur un server windows a partir du Gestionnaire de serveur.

 

Une fois le rôle WDS installer, on doit le configurer pour se faire, on se rend sur l'onglet “WDS” (1), puis on clic droit sur notre server WDS présent dans l'encart server pour choisir “Contrôle de gestion des services de déploiement de windows” (2) .

 

On arrive sur cette page on déroule l'onglet server (1), pour trouver notre serveur, puis on clic droit dessus et on clic sur “Configurer le serveur” (2)

 

On entre dans l'Assistant Configuration des services de déploiement Windows, on commence par cliqué sur suivant (1)

 

Puis on coche “Intégré à Active Directory” (1), et ensuite “Suivant” (2)

 

On arrive sur “Emplacement du dossier d'installation à distance”, pour choisir notre emplacement on clic sur parcourir (1), on choisi notre disque qu'on a crée au préalable, dans lequel on crée un fichier “WDS” (2), puis on peu cliqué sur “OK” (3)

 

On laisse coché ce qui est coché (1) et on clique sur “Suivant” (2)

 

Ici on coche “Répondre à tous les ordinateurs" (1), puis on clique sur “Suivant” (2)

 

On attend l'installation puis on clique sur “Terminer” (1)
