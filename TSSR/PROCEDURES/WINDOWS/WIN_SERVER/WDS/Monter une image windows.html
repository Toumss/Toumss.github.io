Monter une image windows

PRÉ-REQUIS

Pour commencer à monter une image windows, il nous faut un iso du windows que l'on souhaite on l'ouvre avec l'explorateur de fichier, on se rend dans le fichier “sources” (1)

 

Une fois dans le fichier “Sources” on taper “.wim” dans la barre de recherche (1) et on copie dans un dossier à part les fichier “install.cim” (2) et “boot.wim” (3)

INSTALLATION IMAGE

 

On se rend ensuite dans notre service de déploiement dans le dossier Images d'installation on clique droit et on clique su “Ajouter une image d'installation…” (1)

 

Sur cette page on peu renommer le groupe d'image, puis cliqué sur "Suivant" (1)  

 

On arrive sur le choix de l'emplacement du fichier, on clique sur “Parcourir…” (1) puis on va cherche notre fichier “install.wim” (2)

 

On arrive sur l'Assistant Ajout d'images, puis on choisi le version de notre windows (1), puis on clique sur “Suivant” (2)

 

On arrive la page de récapitulatif, on clic sur suivant (1)

 

Une fois la tache terminer on peu cliqué sur “Terminer” (1)

INSTALLATION BOOT

 

Ensuite pour installer le bot on se rend dans notre Service de déploiement Windows puis on clique droit sur “Image de démarrage” et on choisi “Ajouter une image de démarrage” (1)

 

On arrive sur la page de l'emplacement, on clique sur “Parcourir” (1) , puis on séléctionne notre “boot.wim” (2) 

 

Sur cette page on peu changer le nom de l'image, puis cliqué sur “Suivant” (1)

 

On arrive sur la page de récapitulatif, on vérifie et on clique sur “Suivant” (1)

 

Une fois l'installation terminer on clique sur “Terminer” (1)
