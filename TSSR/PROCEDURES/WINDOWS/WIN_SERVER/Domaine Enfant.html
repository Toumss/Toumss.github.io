Domaine Enfant

  1. Pour intégrer un domaine enfant dans un domaine parent on commence par configurer un server normalement (Nom/IP,DNS du domaine parent), puis de redémarrer et de l'insérer dans le domaine parent comme un windows client : “Mofidier” (1), "membre d'un" coché domaine (2) puis choisir le domaine voulu, et enfin “OK” (3) :

 

Il faut ensuite installer le role AD DS sur le server enfant, puis la configurer.
On commence par choisir la 2eme option (1), on choisi 'Domaine enfant" (2), on clic sur “Sélectionner” (3) pour sélectionner le domaine parent, on indique un nom pour notre domaine enfant (4) , puis on peu cliqué sur suivant (5). 

 

(Ce connecter avec l'admin du server parent en cliquant sur “Modifier” et en entrant les identifiant)

Puis cliquer sur “suivant” jusqu'à l'installation du server.
