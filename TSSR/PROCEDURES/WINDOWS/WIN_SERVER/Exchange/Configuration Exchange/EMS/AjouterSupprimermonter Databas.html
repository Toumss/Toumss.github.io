Ajouter/Supprimer/monter Database via EMS

Pour ajouter une base de donnée à Exchange via l'invite de commande “EMS” taper la commande suivante (1) :

New-MailboxDatabase -Server WIN-SERV-EX -Name "TEST1" -EdbFilePath "C:\Program Files\Microsoft\Exchange Server\V15\Mailbox\TEST1\TEST1.edb"

 

Si la commande est accepter on doit voir (2) : “Name” “Server” “Recovery” “ReplicationType” ainsi qu'un message d'avertissement.

 

Pour monter une base de donnée à Exchange via l'invite de commande “EMS” taper la commande suivante :

Mount-Database -Identity Test1

 

Pour supprimer une base de donnée à Exchange via l'invite de commande “EMS” taper la commande suivante (1) :

Remove-MailboxDatabase -Identity Test1

 

 

Si la commande a bien été prise en compte un message demandant de confirmer s'affiche (2), Choisir entre [O], [T], [N] ou [U].
