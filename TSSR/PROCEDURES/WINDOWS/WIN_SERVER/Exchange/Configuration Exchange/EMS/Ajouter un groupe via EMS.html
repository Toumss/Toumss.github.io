Ajouter un groupe via EMS

Pour ajouter un groupe via la console EMS, écrire la commande suivante (1) : 

New-DistributionGroup -Name "Test" -Type "Security/Distribution/Distribution Dynamic"

Si la commande a fonctionner on doit voir “Name" ”Display" “GroupType” “PrimarySmtpAddress” (2)

Pour ensuite ajouter un utilisateur dans un groupe utiliser la commande : 

Add-DistributionGroupeMember -Identity “Test” -Member “Olivier@tssr.local”
