Ajouter utilisateur via EMS

Pour ajouter un utilisateur sur Exchange via EMS se rendre dans l'invite de commande EMS puis taper la commande suivante (1),  :

New-Mailbox -Alias Olivier -Name Olivier -FirstName Olivier -LastName Froger -DisplayName "Olivier Froger" -Password (ConvertTo-SecureString -String 'Greta37!' -AsPlainText -Force) -ResetPasswordOnNextLogon $true -UserPrincipalName "Olivier@tssr.local"

 

 

Si l'utilisateur a bien été crée on voir les lignes “Name” “Alias” “ServerName” “ProhibitSendQuota” s'afficher (2)
