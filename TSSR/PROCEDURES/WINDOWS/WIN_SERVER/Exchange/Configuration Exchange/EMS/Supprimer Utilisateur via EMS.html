Supprimer Utilisateur via EMS

Pour supprimer un utilisateur sur Exchange via EMS se rendre dans l'invite de commande EMS puis taper la commande suivante (1)  :

Remove-Mailbox -Identity “Adrien Restoin”

Si la commande c'est bien exécuter on peux choisir entre "[O] oui [T] Oui pour tout, [N] non, [U] non pour tout (2).
