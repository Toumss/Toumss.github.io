Configurer Server DHCP

Une fois l'option DHCP alloué au server il faut le configurer

 

 1) Commencer par clic sur le drapeau blanc (1), puis “Terminer la configuration DHCP” (2) :

 

2) La fenêtre “Assistant Configuration post-installation DHCP” s'ouvre, Clic sur “Valider” (1) :

 

3)Rendez vous dans les outils “Outils d'administration” (1):

 

4) Puis dans “DHCP” (1) :

 

5) Faire dérouler son server et trouver “IPv4”, clic droit et clic sur “Nouvelle étendu…” (1) :

 

6) “Assistant de nouvelle étendue” s'ouvre, donner un Nom (1) et une Description (2) à son étendue :

 

7) Il faut désormais entrer la plage d'adresse avec une IP de début (1) , une IP de fin(2), la longueur du masque CIDR (3), le masque de sous réseau (4), puis “suivants” (5) :

 

8) Définir une durée de bail (1), puis suivant :

 

9) Coché “oui, je veux configurer ces options maintenant” (1), puis suivant (2) :

 

10) Ajouter l'adresse IP du routeur (paserelle par défaut) (1), et clic “Ajouter”(2), puis “suivant”(3) :

 

11) Pour finir coché “Oui, je veux activer cette étendue maintenant” (1) , puis suivant (2) :

 

Ne pas oublier de mettre le DNS par défaut en IP server (ex :192.168.240.3)
