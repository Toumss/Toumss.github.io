Créer une séquence de tâches pour Windows

Pour utiliser l'image d'installation de Windows 10 de façon à ce qu'elle soit déployée sur une machine, il faut l'ajouter à une séquence de tâches.

On effectue un clic droit sur "Task Sequences "(1)  puis on choisi l'option "New Folder ".(2)

 Cela va permettre de regrouper les tâches pour Windows 10 dans ce dossier.

 

On créez un nouveau dossier nommé "Windows 10 "(1) , puis on clique sur “Next” jsuqu'a la fin de l'opération (2)

 

Quand c'est fait, on effectue un nouveau clic droit cette fois sur notre dossier “Windows 10” (1) , pour cliquer sur "New Task Sequence ".(2)

 

Cette nouvelle page s'ouvre on commence par lui indiquer une ID (1), puis on lui donne un nom (2), puis on clique sur “Next” (3)

 

On laisse en “Standard Client Task sequence” (1) puis on clique sur “Next” (2)

 

Ici on choisi l'image du windows qu'on a importer au préalable (1), puis on clique sur "Next" (2)

 

On ne spécifie pas de clef de licence (1) puis on clique sur “Next” (2)

 

À l'étape "OS Settings ", on indique le nom du compte utilisateur qui sera créé par défaut sur la machine déployée(1) , ainsi que le nom de l'organisation(2) . Ce compte sera administrateur de la machine. Puis on clique sur “Next” (3)

 

À l'étape suivante, on défini un mot de passe pour le compte Administrateur local de cette machine et on le confirme (1) , puis on clique sur “Next” (2)

 

On poursuis jusqu'a la fin puis on clique sur “finish” (1)
