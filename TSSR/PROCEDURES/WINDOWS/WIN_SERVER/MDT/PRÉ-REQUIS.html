PRÉ-REQUIS

Pour commencer il faut q'on télécharge Windows "ADK pour Windows" en utilisant ce lien : https://learn.microsoft.com/fr-fr/windows-hardware/get-started/adk-install#download-the-adk-101253981-september-2023

Une fois lancé on le configure, arrivé sur cette page on choisi les fonctionnalités à installer (1), puis on clique sur installer (2)

 

Si tout c'est bien passé on se trouve sur cette page, plus qu'a cliqué sur “Fermer” (1)

 

 

On passe ensuite à l'add-on Windows PE Windows que l'on trouve sur ce lien : https://go.microsoft.com/fwlink/?linkid=2196224

On commence l'installation on coche (1), puis on clique sur “Suivant” (2)

 

On continu et arrivé sur cette page, on coche (1), puis on clique sur “Installer” (2)

Et on attend la fin de l'installation
