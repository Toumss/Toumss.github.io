Linux c'est quoi ?

 

Histoire de Linux :

 

  • Linux est un système d’exploitation hérité du système UNIX.
  • Il est diffusé sous licence publique générale, une licence libre, et il est gratuit.
  • Sa philosophie : "faire une chose et la faire parfaitement".

 

Linux est diffusé sous licence GPL (General Public Licence). C’est la licence open source la plus utilisée dans le monde ! TOUT le code du système d’exploitation est donc accessible librement et gratuitement.

 

TÉLÉCHARGER UNE DISTRIBUTION LINUX :

 

Linux est un système d’exploitation monolithique modulaire.

Monolithique : Le code de Linux est exécuté dans UN SEUL gros objet en mémoire de l’ordinateur. Le noyau.

Modulaire : Le code du noyau linux est organisé sous la forme de modules. Ces sont des blocs de code.

 

Cette organisation présente un avantage très important ! On peut ajouter et/ou retirer des modules dans un noyau en cours d’exécution, sans avoir à l’arrêter ou à le redémarrer.
