Commandes

cd : Retourne au repertoire source.
cd “…” : Se déplace vers un lieu donné
cd - : Se déplace vers le lieu précédent
cd .. : Se déplace dans le répertoire parent


ls : Liste des lieux accessibles depuis la position actuelle
ls -a : Affiche tous les fichiers du répertoire courant, en incluant les fichier cachés.
ls -l : Affiche la liste des fichiers avec de nombreuse info


pwd : Affiche sa position actuelle

mkdir “…” : Crée un dossier

rm : Supprimé un fichier
rm -r : Supprimé un dossier
 

install openssh-server : Install SSH
ssh “Utilisateur@IP” (ex : ssh tssr@192.168.240.5) : Communiquer avec une autre machine
